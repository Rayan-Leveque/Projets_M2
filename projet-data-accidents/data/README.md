# Données

Données BAAC (accidents corporels de la circulation, data.gouv.fr).

## Sous-dossiers

- `raw/` — fichiers d'origine téléchargés depuis data.gouv.fr (caractéristiques, lieux, véhicules, usagers) pour les années étudiées.
- `utf8/` — fichiers re-encodés en UTF-8 avec séparateur `,` (sortie de la question 1.1).
- `enriched/` — fichiers où les valeurs énumérées sont remplacées par leur libellé (sortie de la question 1.3).

À placer également dans `raw/` : `description-des-bases-de-donnees-onisr-annees-2005-a-2018.pdf`.
